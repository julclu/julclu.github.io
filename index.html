I'm a PhD candidate in the Lupo Lab.

I envision a world where algorithms pervade medicine to enable and improve clinician care. I am especially interested in using machine learning on radiology data to more quickly and accurately diagnose hard-to-treat solid tumors, which could benefit enormously from early detection. Currently, my project is focused on differentiating treatment-induced pseudoprogression from true recurrent glioblastoma using multi-parametric MRI techniques. I am currently a Ph.D. candidate in the Lupo lab with a diverse computational skill set and a deep knowledge of glioma biology and treatments.
